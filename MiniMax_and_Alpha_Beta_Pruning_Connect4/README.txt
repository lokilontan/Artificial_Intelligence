Mini-Max and Alpha-Beta pruning: 
First implemented Mini-Max also then added AB pruning. MonteCarloAI, StupidAI, RandomAI are the pre-created opponents. 

Usage: 
java Main -p1 {AI} -p2 {AI} -text

With -text prints the board into the terminal
Without -text executes the graphic board

java Main : two play with two human players
java Main -p1 {AI} : player 1 is AI player 2 is human

{AI} = MiniMax = AlphaBeta = MonteCarloAI = StupidAI = RandomAI
